## 🤯 Logic Used
- Takes height in **centimeters** and weight in **kilograms** from the user.

- Convert height in meters.

- Applies the formula:
   - BMI = weight/(height(m) * height(m)).

- Classified result into:
   - 🟠 Underweight
   - 🟢 Normal
   - 🟡 Overweight
   - 🔴 Obese
   - 🟤 Extreme Obese
